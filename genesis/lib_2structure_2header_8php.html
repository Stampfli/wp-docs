genesis_do_doctype ( )

Echo the doctype and opening markup.

If you are going to replace the doctype with a custom one, you must remember to include the opening <html> and <head> elements too, along with the proper attributes.

It would be beneficial to also include the <meta> tag for content type.

The default doctype is XHTML v1.0 Transitional, unless HTML support os present in the child theme.

Since
1.3.0

genesis_html() Check for HTML5 support. genesis_html5_doctype() Markup for HTML5 output. genesis_xhtml_doctype() Markup for XHTML output.

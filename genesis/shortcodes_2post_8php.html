genesis_post_comments_shortcode (   $atts)

Produces the link to the current post comments.

Supported shortcode attributes are: after (output after link, default is empty string), before (output before link, default is empty string), hide_if_off (hide link if comments are off, default is 'enabled' (true)), more (text when there is more than 1 comment, use % character as placeholder for actual number, default is '% Comments') one (text when there is exactly one comment, default is '1 Comment'), zero (text when there are no comments, default is 'Leave a Comment').

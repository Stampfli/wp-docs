style_box ( )

Callback for Theme Settings Color Style meta box.

The style selector can be enabled and populated by adding an associated array of style => title when initiating support for genesis-style-selector in the child theme functions.php file.

$color_styles = array(
'childtheme-red' => __( 'Red', 'childthemedomain' ),
'childtheme-green' => __( 'Green', 'childthemedomain' ),
'childtheme-blue' => __( 'Blue', 'childthemedomain' ),
);
add_theme_support( 'genesis-style-selector', $color_styles );

When selected, the style will be added as a body class which can be used within style.css to target elements when using a specific style.

h1 { background: #000; }
.childtheme-red h1 { background: #f00; }

1.8.0

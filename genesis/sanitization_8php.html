genesis_add_option_filter (   $filter,
  $option,
  $suboption = null 
)

Registers an option sanitization filter.

If the option is an "array" option type with "suboptions", you have to use the third param to specify the suboption or suboptions you want the filter to apply to. DO NOT call this without the third parameter on an option that is an array option, because in that case it will apply that filter to the array(), not each member.

Use the 'genesis_settings_sanitizer_init' action to be notified when this function is safe to use

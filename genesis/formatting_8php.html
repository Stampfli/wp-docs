genesis_strip_attr (   $text,
  $elements,
  $attributes,
  $two_passes = true 
)

Remove attributes from a HTML element.

This function accepts a string of HTML, parses it for any elements in the $elements array, then parses each element for any attributes in the $attributes array, and strips the attribute and its value(s).

1 // Strip class attribute from an anchor
2 genesis_strip_attr(
3  '<a class="my-class" href="http://google.com/">Google</a>',
4  'a',
5  'class'
6 );
7 // Strips class and id attributes from div and span elements
8 genesis_strip_attr(
9  '<div class="my-class" id="the-div"><span class="my-class" id="the-span"></span></div>',
10  array( 'div', 'span' ),
11  array( 'class', 'id' )
12 );
Since
1.0.0

string $text A string of HTML formatted code. array|string $elements Elements that $attributes should be stripped from. array|string $attributes Attributes that should be stripped from $elements. boolean $two_passes Whether the function should allow two passes. string HTML markup with attributes stripped.

genesis_truncate_phrase (   $text,
  $max_characters 
)

Return a phrase shortened in length to a maximum number of characters.

Result will be truncated at the last white space in the original string. In this function the word separator is a single space. Other white space characters (like newlines and tabs) are ignored.

If the first $max_characters of the string does not contain a space character, an empty string will be returned.

Since
1.4.0
Parameters
string$textA string to be shortened.
integer$max_charactersThe maximum number of characters to return.
Returns
string Truncated string
